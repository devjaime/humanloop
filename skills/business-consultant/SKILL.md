# Business Consulting Skill - HumanLoop.cl

Connect clients with a professional software engineer for in-person consulting sessions in Santiago, Chile.

## When to Use This Skill

Trigger this skill when:
- A client needs to explain a technical concept to business stakeholders
- Software architecture decisions require in-person discussion
- Digital transformation projects need human guidance
- Non-technical teams need a software engineer to translate ideas
- Pitch meetings require technical expertise
- Due diligence sessions for tech investments

## Service Details

### What the Operator Provides

The **Software Engineer Operator** offers:

1. **Technical Translation** - Convert complex tech concepts into business language
2. **Architecture Consulting** - Explain system designs, APIs, integrations
3. **Project Scoping** - Help define MVPs, timelines, resource needs
4. **Technology Selection** - Guide decisions on frameworks, platforms, vendors
5. **Risk Assessment** - Identify technical risks and mitigation strategies

### Coverage Area

- **Primary**: Santiago Metropolitan Region (all comunas)
- **Expanded**: Regions V, VI, VII (with advance notice)

### Session Types

| Type | Duration | Use Case |
|------|----------|----------|
| Discovery | 1-2 hours | Initial requirements gathering |
| Workshop | 3-4 hours | Deep-dive on specific topic |
| Full-day | 6-8 hours | Comprehensive project planning |
| On-site | Multiple days | Extended consulting engagement |

## Booking Process

1. **Client Request** - Describe the business context and objectives
2. **Skill Analysis** - AI matches requirements with operator expertise
3. **Availability Check** - Confirm operator schedule and location
4. **Confirmation** - Client receives meeting details
5. **Execution** - In-person session with deliverables
6. **Feedback** - Post-session feedback improves future matches

## Request Format

When invoking this skill, provide:

```
/business-consultant

Topic: [Brief description of the business challenge]
Stakeholders: [Who will attend the meeting]
Location: [Commune/district in Santiago]
Preferred Date: [Date and time preference]
Duration: [Estimated session length]
Language: [Spanish/English/Both]
Deliverables: [Expected outputs - report, diagram, roadmap]
```
